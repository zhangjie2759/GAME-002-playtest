# GAME-002｜攻防战 H5 v0.4 基础塔防原型

这是降低难度后的基础塔防版本。目标不是验证复杂双区攻防，而是先做出一个能玩的塔防底座。

## 当前目标

* 敌兵从上方入口沿通道进攻我方城门。
* 玩家拖底部塔牌到高地建塔。
* 塔自动攻击范围内敌兵。
* 敌兵到达城门会扣血。
* 守住 6 波敌兵则胜利。
* 城门血量归零则失败。

## 操作

* 拖动底部卡牌到发亮的“高地”。
* 高地空着时会建塔。
* 高地已有塔时不能重复建塔。
* 手牌会随时间补充，最多 3 张。

## 塔

* 盾塔：近距离稳定防守。
* 枪塔：中距离高伤害。
* 弓塔：远距离持续射击。

## 验收标准

这一步只要求“基础塔防成立”：

* 能完整玩一局。
* 能建塔。
* 敌人会沿路进攻。
* 塔会自动攻击。
* 能赢也能输。
* 第一眼能看懂这是一个手机塔防。
