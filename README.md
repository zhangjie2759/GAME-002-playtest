# GAME-002｜攻防战 H5 v0.4B 基础塔防原型

这是降低难度后的基础塔防版本。目标不是验证复杂双区攻防，而是先做出一个能玩的塔防底座。

## 当前目标

* 敌兵从上方入口沿通道进攻我方城门。
* 玩家拖底部塔牌到高地建塔。
* 塔自动攻击范围内敌兵。
* 敌兵到达城门会扣血。
* 守住 6 波敌兵则胜利。
* 城门血量归零则失败。

## 操作

* 拖动底部卡牌到发亮的“高地”。
* 高地空着时会建塔。
* 高地已有塔时不能重复建塔。
* 手牌会随时间补充，最多 3 张。
* 击退敌兵会获得军资。
* 点击已经建好的塔可以消耗军资升级，最多 3 级。
* 点击右上角“齐射”可以让所有塔立刻准备开火，有冷却时间。

## 塔

* 盾塔：近距离减速，适合放在转角拖住敌兵。
* 枪塔：中距离高伤害。
* 弓塔：远距离持续射击。

## v0.4A 修正

* 重排高地位置，让塔位更贴近入口、转角和终点前关键路段。
* 盾塔改为高频近防减速塔，不再只是低伤害短射程塔。
* 新增军资：击退敌兵获得军资。
* 新增塔升级：点击已建塔可消耗军资升级。
* 新增齐射：点击右上角按钮让所有塔立刻准备开火，增加波次中的操作。

## v0.4B 修正

* 开局手牌顺序改为盾、枪、弓，让盾塔更早进入测试。
* 盾塔减速从 0.48 强化为 0.38，持续时间从 1.15 秒增加到 1.5 秒。
* 一级升级成本从 5 军资降为 4 军资，建塔后可以立即做一次主动强化选择。

## 验收标准

这一步只要求“基础塔防成立”：

* 能完整玩一局。
* 能建塔。
* 敌人会沿路进攻。
* 塔会自动攻击。
* 能赢也能输。
* 第一眼能看懂这是一个手机塔防。
